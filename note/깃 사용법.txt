
# GIT

 - .git 폴더가 위치한 폴더의 모든 내용을 관리해주는 프로그램
 - .git 폴더를 생성한 곳을 Repository라고 부른다
 - .git 폴더 내부에는 관리하는 Repository의 모든 정보가 기록되어 있다
 - .git에 저장된 기록을 활용하여 관리하는 Repository를 예전 모습으로 되돌릴 수 있다
 - GIT은 Repository 내부의 무언가가 아주 약간만 변해도 모두 알아차리고 관리해준다
 - 이클립스에는 Egit이라는 프로그램이 내장되어 있다

# 이클립스에서 git 사용

이클립스 -> window -> show view -> others -> Git -> Git Repositories
CREATE(3개 중 마지막) -> 저장할 곳 선택 -> 자바 프로젝트 생성 중 경로를 이곳으로 설정
